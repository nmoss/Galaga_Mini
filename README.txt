README.txt

Files needed to play the game:
	galaga.html
	background.js
	galaga_sprite.js

Image files needed:
	background.png
	enemy_missile.png
	fighter.png
	galagan_blue_animated.png
	galagan_explosion.png
	missile.png
	red_animation.png
	ship_explosion.png
	sprites.png

Put all of these files in the same directory, and then load the game into Google Chrome. 
The game uses a Chrome specific keyboard event handler, which may or may not be standard in other browsers.
